## 在线编辑

https://replit.com/@Jeff_Tian/PearlsPlus

要运行一个指定的程序文件，可以在 Shell 里使用如下命令：

```shell
clang++-7 -o main chapter1/1.6.1.cpp
./main
```

如果是 c 文件，则要用这个命令：

```shell
clang-7 -o main chapter1/1.6.1.c
./main
```
